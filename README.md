# tinyhouse backend

React Fullstack App - Backend
Node/Express Server

## Start Server

1. `cd` into `server` folder.
2. Execute `npm run start`.

## Seed Mock Data in DB

1. `cd` into `server` folder.
2. Execute `npm run seed`.

## Build the App for Production

1. `cd` into `server` folder.
2. Execute `npm run build`.
